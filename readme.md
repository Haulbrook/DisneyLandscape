## ğŸ¯ Disney Design Standards

This tool enforces Disney Imagineering landscape principles:

| Rule | Standard |
|------|----------|
| Coverage | 95%+ living material, 5% max visible mulch |
| Spacing | Front: 6-8", Middle: 12-18", Back: 24-36" |

| Perspective | Height graduation front-to-back |
| Focal | Every bed needs a "weenie" (focal point) |
